## Instructions

Switch to your branch before starting to code <br>

1. Open a terminal and run the following command:
```bash
  cd backend
  npm i
  npm run start
```
2. Open another terminal and run the following command:
```bash
  cd frontend
  npm i
  npm run dev
```
